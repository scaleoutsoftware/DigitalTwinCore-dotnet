AddAnomalyDetectionProvider(RealTimeWorkbench, string, string, string, ILogger)

This reads the Zip file produced by the ScaleOut Machine Learning Training Tool to create a new instance of IAnomalyDetectionProvider so that the RealTimeProcessingContext has access to it. It allows digital twin instances to call the DetectAnomaly API through the processing context.
